
## Basic Operations 

## Show Loaded Modules:
```bash
module list
```

## Show Available Modules:
```bash
module avail
```

## Load a Module:
```bash
module load <module>
```

## Unload a Module:
```bash
module unload <module>
```

## Unload All Modules:
```bash
module purge
```

## Brief Description of a Module:
```bash
module whatis <module>
```

## Display Help for a Module:
```bash
module help <module>
```
